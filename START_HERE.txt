## 🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: "ModuleNotFoundError"
**Решение:**
```bash
pip install -r requirements.txt
```

### Проблема: "BOT_TOKEN is not set"
**Решение:**
1. Открыть .env файл
2. Добавить: BOT_TOKEN=ВАШ_ТОКЕН
3. Сохранить
4. Перезапустить

### Проблема: Нет подключения к БД
**Решение:**
1. Убедиться что PostgreSQL запущена
2. Проверить DATABASE_URL в .env
3. Проверить логи (они выводятся в консоль)

### Проблема: Проекты не открываются
**Решение:**
- Обновлены все файлы в v2.0
- Ошибка должна быть исправлена
- Если нет - перезагрузи бот

### Проблема: Фото не загружаются
**Решение:**
- Обновлены обработчики фото
- Должно работать теперь
- Если нет - проверь логи

### Проблема: Кнопка "Назад" не работает
**Решение:**
- Добавлена везде в v2.0
- Должна работать везде
- Если не видишь - прокрути вниз
